This parameter switches the default AWS API mode to AWS CLI mode with use of AWS S3 Command Line Interface (CLI) file commands for artifact files upload to S3.

This option is a workaround for big files transfer API error for file bigger than 5Gb (MaxSizeAllowed=5368709120 bytes):

ERROR: Step ‘Archive the artifacts’ failed ..., response: 400 Bad Request,
 EntityTooLarge: Your proposed upload exceeds the maximum allowed size
